# 参照キャプチャ：2026-07-01 ch17 カンテレ（壁アンテナ）

壁のTVアンテナ端子（F→SMA変換）から受けた、C/Nに余裕のあるゴールデンデータ。
弱アンテナ版（2026-06-28, g49.6）で当たった**C/Nの壁を突破**した本命キャプチャ。

- ファイル：`2026-07-01_ch17_kantele_497142857_fs1015873_g30_wall.iq`
- フォーマット：rtl_sdr 生IQ（u8 インターリーブ I,Q）
- 受信：壁のTVアンテナ端子 ＋ F→SMA変換
- 物理ch：17ch ＝ 関西テレビ（カンテレ）
- 中心周波数：`497142857 Hz`
- サンプルレート：`1015873 Hz`
- **ゲイン：30 dB**（壁アンテナは信号が強く、最大49.6だとADC飽和するため下げた。clip 0.01%, std 33.7 で健全）
- 長さ：約13.3 s（27 MB）

## 観測結果（本キャプチャで到達）

| 段 | 結果 |
|----|------|
| ② 同期 | ロック（GI=1/8） |
| ③ 等化 | QPSK、クラスタ明瞭 |
| ④ TMCC | 同期100% / Layer A = QPSK 2/3 I=4 1seg |
| ⑤ FEC | **Viterbiロック（再エンコード一致率 1.000 vs 雑音床0.928）**。設定 order=1, shift=0, pphase=0 |
| ⑥ TS | **同期バイト 0x47 が204バイト周期で命中率100%（TSパケット同期を確認）** |

再現：
```bash
# キャプチャ（壁アンテナ）
timeout 14 rtl_sdr -f 497142857 -s 1015873 -g 30 cap.iq
# FECロック確認
cargo run -p isdbt-dsp --example fec_lock_probe -- cap.iq 1015873 2400
# TSパケット同期
cargo run -p isdbt-dsp --example ts_probe -- cap.iq 1015873 3000
```
